Realm支持的操作符（OC）

发表于 2017-03-30   |     |   阅读次数

运算符

比较操作数可以是属性名称或者常量。至少有一个操作数必须是属性名称。

  • int, long, longlong, float, double和NSDate属性支持比较运算符 ==，<=, <, >=， != 和 BETWEEN。 如 age == 45

  • identify比较 ==，!=, 例如 [Employee objectsWhere:@"company == %@",company]

  • 布尔属性支持比较运算符==，!=

  • 对于NSString，NSData属性，支持==，!=, BEGINSWITH, CONTAINS, ENDSWITH 运算符。如 name CONTAINS 'Ja'

  • 为NSString属性时，可以使用LIKE操作符，*和?作为通配符，其中?匹配一个字符，*匹配多个字符。如 value LIKE ‘?bc*’匹配字符串，匹配到的结果可以是“abcde”,“cbc”

  • 不区分大小写字符串比较[c]，如 name CONTAINS[c] 'Ja'

  • 区分大小写的字符串比较[d],如 name BEGINSWITH[d] 'e'

  • Realm 支持以下复合运算符：AND,OR和NOT,如name BEGINSWITH 'J' AND age > 32

  • 容器运算符 IN,如 name IN {'Lisa','Spike','Hachi'}

  • Nil 比较运算符 ==,!=,如 [Company objectsWhere:"ceo == nil"],

  • ANY 运算符，如 ANY student.age < 21

  • RLMArray 和 RLMResults 支持集合公式@count,@min,@max,@avg,如[Company objectsWhere:@"employess.@count > 5"],查找所有的公司有超过5名员工的
