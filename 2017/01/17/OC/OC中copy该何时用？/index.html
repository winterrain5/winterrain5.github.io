OC中copy该何时用？

发表于 2017-01-17   |   分类于 Objective-C   |     |   阅读次数

值对象

指封装了基本值（属于C数据类型）且提供与该值相关的服务的对象。值对象以对象形式表示标量类型。

Foundation框架向您提供了以下类：

  • NSString和NSMutableString

  • NSData和NSMutableData

  • NSDate

  • NSNumber

  • NSValue

理解

  • 深拷贝（copy）： 只复制指向对象的指针，而不复制引用对象本身。通过对象的指针来访问这个对象。
  • 浅拷贝（mutableCopy）： 复制引用对象本省

也就是说有个A对象，复制一份后得到A_Copy对象，对于浅复制来说，A和A_Copy指向的是同一个内存资源，复制的只不过是一个指针，对象本身资源还是只有一份。如果我们对A_Copy执行了修改操作，发现A引用的对象同样被修改。而深复制可以理解为内存中存在了两份独立的对象本身，当A修改时，A_Copy不变。
