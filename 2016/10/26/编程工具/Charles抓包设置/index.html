Charles抓包设置

发表于 2016-10-26   |     |   阅读次数

HTTP 抓包

查看电脑IP

设置手机代理

手机连上电脑，设置->无线局域网->连接的WiFi，设置HTTP代理：
服务器为电脑IP地址：192.168.1.122
端口： 8888

电脑打开Charles进行HTTP抓包

手机上打开某个APP后，Charles会弹出选项框

点击允许，出现手机的HTTP请求列表

HTTPS抓包

在HTTP设置的基础上需要另行设置才能抓到HTTPS的包

设置之前是长这样的

可以看到包内容是乱码

安装SSL证书到手机设备

在Charles 点击 Help->SSL Proxying -> Install Charles Root Certificate on a MObile Device

出现弹窗，在Safari输入地址 http://Charlesproxy/getssl，然后一路安装即可

Charles设置Proxy

Proxy -> SSL Proxying Settings

勾选Enable SSL Proxying,点击Add

输入你要抓取的HTTPS接口

之后你便可看到成功抓取HTTPS包的内容

不抓包后记得关闭HTTP代理，否则断开和电脑的连接后会连不上网络
