JavaScript访问关系

发表于 2016-11-21   |   分类于 前端   |     |   阅读次数

父节点

  • parentNode

兄弟节点

  • nextSibling 下一个兄弟 只有IE6、7、8识别
  • nextElementSibling 其他浏览器识别
  • previousSibling 上一个兄弟 只有IE6、7、8识别
  • previousElementSibling 其他浏览器识别

要想兼容所有浏览器，可以这样写：

var div = document.getElementById("div");
var next = div.nextElementSibling || div.nextSibling

必须先写其他浏览器，否则会报错。

子节点

  • firstChild 第一个子节点
  • firstElementChild
  • lastChild 最后一个子节点
  • lastElementChild

所有子节点

  • childNodes

返回指定元素的子元素集合，包括HTML节点、文本节点等。火狐、谷歌等高版本会把换行也看做子节点
一般情况下，我们只需要元素节点,可以通过nodeType来判断： 

nodeType == 1    元素节点     
nodeType == 2    属性节点
nodeType == 3    文本节点
  • children 返回指定元素的标签节点。IE6、7、8会把注释节点也算进去
