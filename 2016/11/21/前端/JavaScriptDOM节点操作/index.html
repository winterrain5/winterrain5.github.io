DOM定义

DOM为文档提供了结构化表示，并定义了如何通过脚本来访问文档结构。目的其实是为了让js操作HTML元素而制定的一个规范。

DOM树

节点

由结构图我们可以看到，整个文档就是一个文档节点，而每个HTML标签都是一个元素节点。标签中的文字则是文字节点。标签的属性是属性节点。总之，一切都是节点。。。。

各节点的相互关系

创建节点

  • createElement()

    var div = document.createElement("div");

插入节点

  • appendChild() 添加到最后

    var fatherDiv = document.getElementById("father");
var childDiv = document.createElement("div");
fatherDiv.appendChild(childDiv);

  • insertBefore() 添加节点到某个节点的前面

    var childrens = fatherDiv.children();
var firstChildDiv = document.createElement("div");
// 插入到 fatherDiv的第一个位置 第二个参数null，则默认放到最后
fatherDiv.insertBefore(firstChildDiv,childrends[0]);

移除节点

  • removeChild() 移除孩子节点 

    var child = document.getElementById("child");
fatherDiv.removeChild(child);

克隆节点

  • cloneNode()

参数为true，为深层复制，除了复制本盒子还复制子节点

参数为false，为浅层复制，只复制本节点，不复制子节点
